Echo

本地启动、内网访问、多人同步批注，适合产品 HTML demo 和需求评审。

一、启动

  1. Windows 双击 startup-methods/windows-echo.bat；macOS 双击 startup-methods/macos-echo.command。
  2. 等待浏览器自动打开 http://localhost:5177。
  3. 在控制台导入 HTML 文件，或输入一个可访问链接保存页面快照。
  4. 选择同事能访问的地址，再复制评审链接发给同事。

二、评审

  • 点击“评论”，再点击页面元素添加批注。
  • 点击批注圆点，查看该位置的批注记录。
  • 点击右侧批注卡片，定位回原区域并高亮闪烁。
  • 切换“需求”模式后，可以为选中区域编写需求说明。

三、结束

Windows 使用 startup-methods/windows-echo-stop.bat 停止服务；macOS 使用 startup-methods/macos-echo-stop.command 停止服务。

四、给同事访问

同事必须和你在同一内网或 VPN。请在控制台选择同事能访问的 IP，再发送评审链接，例如 http://192.168.x.x:5177/review/...，不要发送 localhost 链接。
